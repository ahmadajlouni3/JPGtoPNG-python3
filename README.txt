About:

welcome to simple images converter from jpg to png with size enhancer

Usage:

when you run the program you must put arg like this:

windows:

	py {pythonFile.py} {the folder which have jpg images} {the folder you want to create to send png images}

linux/mac:

	python3 {pythonFile.py} {the folder which have jpg images} {the folder you want to send png images}


only put any jpg images into images folder and run the program. after that the program will convert<br/>
all images and send them to new_image folder.
